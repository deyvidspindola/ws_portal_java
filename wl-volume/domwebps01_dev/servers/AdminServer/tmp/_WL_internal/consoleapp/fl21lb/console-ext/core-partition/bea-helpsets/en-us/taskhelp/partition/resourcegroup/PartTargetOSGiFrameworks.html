Target OSGi frameworks


Assign servers or clusters for an OSGi framework. The OSGi framework is deployed on the server(s) or cluster(s) that you select.

To target OSGi frameworks:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the left pane, expand Services and select OSGi Frameworks.

    A table listing the OSGi frameworks that have been created appears.
  3. Click the name of the OSGi framework that you want to assign to a server or cluster.
  4. On the Targets page:
    1. Select the checkbox(es) next to the server(s) or cluster(s) to which you want to assign to the OSGi framework.
    2. To remove an OSGi framework from a server or cluster, deselect the checkbox next to the server or cluster from which you want to remove the OSGi framework.
  5. Click Save.
  6. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
