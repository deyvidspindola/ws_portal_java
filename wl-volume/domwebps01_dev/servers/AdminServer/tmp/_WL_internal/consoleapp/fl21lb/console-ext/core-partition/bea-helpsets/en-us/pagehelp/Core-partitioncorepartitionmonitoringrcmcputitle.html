Domain Partition: Monitoring: Resource Management: CPU Utilization

Use this page to monitor the CPU Utilization for a partition, including associated triggers and constraints.

Configuration Options

Name Description
CPU Utilization

The percentage of CPU time utilized.
Fairshare Constraint Name

The name of the fair share constraint.
Fairshare Constraint Value

The amount allocated (a percentage of the available resource) for a fairshare constraint.

Column Display

You can show fewer or additional data points on this page by expanding Customize this table and modifying the Column Display list. Each data point displays in its own table column.

The following table lists all of the data points that you can display in columns on this page.

Name Description
Trigger Name

Name of the trigger.
Trigger Value

A static constraint value specifying the amount of resource allocated for a trigger.
