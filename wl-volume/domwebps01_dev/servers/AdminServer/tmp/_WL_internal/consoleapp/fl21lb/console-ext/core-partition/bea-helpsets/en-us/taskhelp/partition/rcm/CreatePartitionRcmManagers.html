Create a partition resource manager


To create a resource manager specifically for this partition:

  1. In the left pane of the Console, expand Domain Partitions.

    The Domain Partitions table displays all the partitions defined in the domain.

  2. Click on the name of the domain partition where you want to create a resource manager.
  3. Select Configuration: Resource Management.
  4. Select NONE from the Resource Manager Reference drop-down box.
  5. Select Resource Manager Name: Change to create a resource manager that is specific for this domain partition.
    1. Name - The name of the resource manager to create.
    2. Scope - Specifies the domain patition where the resource manager is accessible.
    3. Notes - Provide additional information that describes the configuration of this resource manager.
  6. Click Save.

After you finish

Select Resource Manager Name: Change and Manage Resource Conumption Manager Policies.
