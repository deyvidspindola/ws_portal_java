Domain Partition: Configuration: Resource Management

Configuration Options     Related Tasks     Related Topics

Use this page to define the resource manager for a domain partition. A domain partition should have either a reference to a domain resource manager or partition specific resource manager.

Configuration Options

Name Description
Resource Manager Reference

A resource manager reference from the resource management.

MBean Attribute:
PartitionMBean.ResourceManagerRef

Resource Manager Name

The resource manager policy assigned to the partition.

MBean Attribute:
PartitionMBean.ResourceManager
