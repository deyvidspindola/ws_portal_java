Use this page to view running batch jobs in this domain partition.

Column Display

You can show fewer or additional data points on this page by expanding Customize this table and modifying the Column Display list. Each data point displays in its own table column.

The following table lists all of the data points that you can display in columns on this page.

Name Description
Job Name

The name of the running batch job.
Application Name

The name of the application that submitted the job.
Instance ID

The instance ID of the job.
Execution ID

The execution ID of the job.
Batch Status

The batch status of the job.
Start Time

The start time of the job.
End Time

The completion time of the job.
Exit Status

The exit status of the job.
