Configure partition resource managers

Before you begin

Create a Resource Consumption Manager


Resource Consumption Management (RCM) provides a flexible, dynamic mechanism to manage shared resources and provide consistent performance of domain partitions. Use this page to select an existing domain resource manager or to create a new resource manager for this domain partition.

To configure a Partition Resource Manager:

  1. In the left pane of the Console, expand Domain Partitions.

    The Domain Partitions table displays all the partitions defined in the domain.

  2. Click on the name of the domain partition where you want to configure a resource manager.
  3. Select Configuration: Resource Management.
  4. Select an existing resource manager from the Resource Manager Reference drop-down box. To create a domain partition specific resource manager, select NONE.
  5. If you selected NONE from the Resource Manager Reference drop-down box, select Resource Manager Name: Change to create or manage a resource manager that is specific for a domain partition.
    1. To create a new resource manager for this domain partition, see Create a partition resource manager.
    2. For an existing resource manager, select Resource Manager Name: Change and Manage Resource Conumption Manager Policies.
