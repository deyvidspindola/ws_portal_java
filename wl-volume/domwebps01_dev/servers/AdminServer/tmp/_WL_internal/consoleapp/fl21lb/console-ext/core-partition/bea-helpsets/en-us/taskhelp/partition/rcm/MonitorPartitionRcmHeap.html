Monitor heap utilization

Before you begin

Configure partition resource managers


To monitor heap utilization for this domain partition:

  1. In the left pane of the Console, expand Domain Partitions.

    The Domain Partitions table displays all the partitions defined in the domain.

  2. Click on the name of the domain partition that you want to monitor.
  3. Select Monitoring: Resource Management: Heap Retained.

Result

The monitoring information for heap utilization in this domain partition is displayed, including associated trigger and fair share constraint status.
