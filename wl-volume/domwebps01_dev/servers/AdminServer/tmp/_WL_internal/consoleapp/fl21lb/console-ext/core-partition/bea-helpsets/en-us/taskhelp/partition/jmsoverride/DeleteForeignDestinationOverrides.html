Delete foreign destination overrides


To delete a foreign destination override:

  1. In the left pane of the Console, select Domain Partitions.

    The Domain Partitions table displays all the partitions defined in the domain.

  2. Click on the name of the domain partition from which you want to delete a foreign destination override.
  3. Select the Resource Overrides > JMS Modules configuration page.

    The JMS Module Overrides table displays all the JMS module overrides that have been configured in this domain partition.

  4. Click on the name of the JMS module override from which you want to delete a foreign destination override.
  5. Click on the name of the JMS foreign server override from which you want to delete a foreign destination override.
  6. On the Configuration > Destination Overrides page, select the check box next to the name of the foreign destination override you want to delete.
  7. Click Delete.

After you finish

You must restart the partition for the override removal to take effect.
