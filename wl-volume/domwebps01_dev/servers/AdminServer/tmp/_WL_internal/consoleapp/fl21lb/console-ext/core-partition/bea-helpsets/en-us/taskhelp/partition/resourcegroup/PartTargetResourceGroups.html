Target resource groups at the partition level to virtual targets


You can choose the virtual targets for the resource group from the virtual targets that are available to the partition.

To target a resource group at the partition level to a virtual target:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the left pane of the Console, select Domain Partitions.
  3. On the Summary of Domain Partitions page, select the partition for which you want to target the resource group.
  4. Select the Resource Groups page.
  5. Select the resource group you want to target.
  6. Set the Use Default Target control to use the default target configured for the partition. Alternately, select one or more virtual targets in the Available list and click the arrow to move the virtual targets to the Chosen list.

    The Console does not require you to have a virtual target for the resource group. However, you will not be able to start the resource group if at least one virtual target does not appear in the Chosen list.

  7. Click Save.
  8. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
