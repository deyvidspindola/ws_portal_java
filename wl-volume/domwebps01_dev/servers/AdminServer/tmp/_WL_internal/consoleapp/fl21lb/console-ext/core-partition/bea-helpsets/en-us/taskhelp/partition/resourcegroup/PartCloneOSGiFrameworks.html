Clone OSGi frameworks


WebLogic Server allows you to add a list of OSGi frameworks to the server configuration.

To clone an OSGi framework:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the left pane, expand Services and select OSGi Frameworks.

    A table listing the OSGi frameworks that have been created appears.
  3. Select the checkbox next to the OSGi framework that you want to clone.
  4. Click Clone.

    A dialog displays asking for the name of your new OSGi framework.
  5. Enter the name of your OSGi framework in the Name field.
  6. Click OK.

    The new instance is added in the OSGi Frameworks table.
  7. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
