Create a JMS System Module Override

Configuration Options     Related Tasks     Related Topics

Use this page to create a JMS system module override.

Configuration Options

Name Description
Name

The name of this JMS module override. It must match the name of the actual resource to be overridden.

MBean Attribute:
JMSSystemResourceOverrideMBean.Name

Changes take effect after you redeploy the module or restart the server.
