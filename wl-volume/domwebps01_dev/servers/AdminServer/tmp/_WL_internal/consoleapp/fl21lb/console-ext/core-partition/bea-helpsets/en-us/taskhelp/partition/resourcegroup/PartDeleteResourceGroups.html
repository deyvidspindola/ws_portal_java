Delete resource groups at the partition level


Deleting a resource group permanently removes it from the partition's configuration. When you delete a resource group, all of the applications deployed to the resource group are undeployed.

To delete a resource group at the partition level:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the left pane of the Console, select Domain Partitions.
  3. Select the partition for which you want to delete the resource group.
  4. Select the Resource Groups page.
  5. On the Configuration page, select the partition you want to delete and click Delete.
  6. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
