Delete mail session overrides


To delete a mail session override:

  1. In the left pane of the Console, select Domain Partitions.

    The Domain Partitions table displays all the partitions defined in the domain.

  2. Click on the name of the domain partition from which you want to delete a mail session override.
  3. Select the Resource Overrides > Mail Sessions configuration page.

    The Mail Session Overrides table displays all the mail session overrides that have been configured in this domain partition.

  4. Select the check box next to the name of the mail session override you want to delete.
  5. Click Delete.

After you finish

You must restart the partition for the override removal to take effect.
