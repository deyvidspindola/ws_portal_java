Create OSGi frameworks


The WebLogic Server OSGi service allows you to add a list of OSGi frameworks to the domain configuration.

To create an OSGi framework:

  1. If you have not already done so, in the Change Center of the Administration Console, click Lock & Edit (see Use the Change Center).
  2. In the left pane, expand Services and select OSGi Frameworks.

    A table listing the OSGi frameworks that have been created appears.
  3. Click the New button.
  4. Enter the name of your framework in the Name field.
  5. Click Next.
  6. Select the servers or clusters to which you would like to deploy this OSGi framework.
  7. Click Finish.

    The new OSGi framework is added in the OSGi Frameworks table.
  8. In the OSGi Frameworks table, click the name of the framework you just created.
  9. Use the Configuration page to modify the configuration of an OSGi framework. See OSGi Frameworks: Configuration.
  10. Click Save.
  11. Use the Targets page to select servers or clusters on which to deploy your OSGi framework. See Target OSGi frameworks.
  12. Use the Notes page to enter notes related to the framework.
  13. Click Save.
  14. To activate these changes, in the Change Center of the Administration Console, click Activate Changes.
    Not all changes take effect immediately—some require a restart (see Use the Change Center).
