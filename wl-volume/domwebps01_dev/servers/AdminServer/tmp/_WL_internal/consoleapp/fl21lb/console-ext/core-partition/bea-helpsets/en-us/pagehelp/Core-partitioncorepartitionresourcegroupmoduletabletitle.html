Diagnostic System Modules

Column Display     Related Tasks     Related Topics

This page summarizes the diagnostic system modules that have been configured in the current partition resource group.

A diagnostic system module is a software component that you configure to monitor an aspect of a server or server resource.

For example, you can configure a diagnostic system module to monitor how system-wide JDBC data sources use memory. You can configure module to monitor the number of open sockets on a server and the number of connections used in connection pools. You can then activate the modules on servers as needed to diagnose problems. You can also dynamically activate or deactivate a module without making a change to the domain configuration.

While you can use the Administration Console to configure and deploy diagnostics system modules, you cannot use the Administration Console to do the same for diagnostic modules for applications. Diagnostic modules for applications are configured through a deployment descriptor and then packaged with the application archive and deployed with an application.

Column Display

You can show fewer or additional data points on this page by expanding Customize this table and modifying the Column Display list. Each data point displays in its own table column.

The following table lists all of the data points that you can display in columns on this page.

Name Description
Name

The name of this diagnostics system module.

MBean Attribute:
WLDFSystemResourceMBean.Name

Changes take effect after you redeploy the module or restart the server.

Description

The description for this diagnostics system module.

MBean Attribute:
WLDFSystemResourceMBean.Description

Targets

The server or cluster to which this diagnostic system module is targeted.

You can specify one or more servers and one or more clusters as the targets for a diagnostic system module.
Scope

The scope in which this diagnostics system module is deployed, which can be either global or partition-specific.
Domain Partitions

List of the domain partitions in which this diagnostics system module is available.
